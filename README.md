# スピードランナーVerコインゲッター
## ゲーム概要
破壊可能なオブジェクトをすべて破壊してゴールを目指す3Dソードアクションゲーム。近距離の剣攻撃と遠距離のスラッシュ攻撃を使い分け、タイムを競う。

## 操作方法
WASD / 矢印キー：移動

左クリック：近距離攻撃（剣）

右クリック：遠距離攻撃（スラッシュ）

Space：ジャンプ

マウス：カメラ回転

## 使用技術
・Unity 2023.2.19f1

・C#

## 工夫した点
近距離攻撃（剣のコライダー判定）と遠距離攻撃（プロジェクタイル生成）の2種類の攻撃システムを実装。

リアルタイムで計測したクリアタイムを PlayerPrefs に保存し、ベストタイムとして次回以降に表示する仕組みを実装。

Physics.CheckSphere を用いた接地判定により、ジャンプおよび攻撃を地上時のみ可能にする制御を実装。

逆メッシュコライダーを使用したステージ境界システムにより、見えない壁でプレイヤーをステージ内に閉じ込める処理を実装。
